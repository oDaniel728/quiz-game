# `resource_location`
São utilizados pra se referir a arquivos de um datapack ou de outros datapacks.

## Formato geral
### `datapack:nome`
Sendo `datapack` a pasta em `/data/` e `nome` sendo um arquivo de uma pasta no datapack, a gente vai chamar de `contexto`.

# `resource_location<Event>`
São utilizados pra se referir a arquivos json no contexto eventos.

### Exemplo
`game:success` é o arquivo `/data/game/events/success.json`

# `resource_location<Question>`
São utilizados pra se referir a arquivos json no contexto questions(perguntas)

### Exemplo
`game:1` é o arquivo `/data/game/questions/1.json`
